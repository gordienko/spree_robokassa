SpreeRobokassa
==============

Installation
------------

Add spree_robokassa to your Gemfile:

```ruby
gem 'spree_robokassa', github: 'gordienko/spree_robokassa'
```

Bundle your dependencies

```shell
bundle install
```

Run install

```shell
bundle exec rails g spree_robokassa:install
```
